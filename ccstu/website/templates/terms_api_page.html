ข้อตกลงการอนุญาตใช้งาน API

    ข้อตกลงการอนุญาต API นี้ (ซึ่งต่อไปนี้เรียกว่า "ข้อตกลง") กำหนดสิทธิ์การเข้าถึงของผู้ใช้งาน (ซึ่งต่อไปนี้เรียกว่า "ผู้รับอนุญาต") สำหรับ API ข้อมูล และเอกสารประกอบที่จัดหาให้และให้อนุญาตโดย CCS Thammasat การใช้ API ของ CCS Thammasat จะถือว่าผู้ใช้งานอยู่ภายใต้ข้อผูกมัดในการปฏิบัติตามเงื่อนไขของข้อตกลงนี้ หากผู้ใช้งานไม่ยินยอมต่อเงื่อนไขของข้อตกลงนี้ จะไม่สามารถใช้ API ได้

  1. ข้อกำหนดทั่วไป
    • "อินเทอร์เฟซการเขียนโปรแกรมแอปพลิเคชัน" หรือ "API" เป็นชุดของไลบรารี เครื่องมือ ตัวอย่างโค้ดต้นฉบับ ข้อมูลจำเพาะที่เผยแพร่ และเอกสารประกอบที่จัดทำโดย CCS Thammasat ทั้งนี้ CCS Thammasat อาจจัดให้มีการปรับปรุงหรือการเพิ่มเติม API สำหรับผู้รับอนุญาต ภายใต้ดุลยพินิจของตนเอง
  2. การอนุญาต
    • ภายใต้เงื่อนไขของข้อตกลงฉบับนี้ CCS Thammasat มอบสิทธิ์ที่จำกัด ไม่ใช่สิทธิ์ขาดแต่เพียงผู้เดียว และถ่ายโอนไม่ได้ (ไม่มีสิทธิ์ในการให้สิทธิ์ต่อ) แก่ผู้รับอนุญาตในการใช้ API เพื่อการใช้งานส่วนตัวของผู้รับอนุญาตแต่เพียงประการเดียว
    • ผู้รับอนุญาตไม่อาจแจกจ่าย ให้อนุญาต หรือโอนสิทธิ์ API ในทางอื่นใดให้แก่บุคคลภายนอกได้
  3. สิทธิ์และข้อจำกัดอื่นๆ
    • ผู้รับอนุญาตสามารถคัดลอก API ได้เพื่อการใช้งานภายใต้เงื่อนไขของข้อตกลงฉบับนี้เท่านั้น
    • การทำวิศวกรรมย้อนกลับ ผู้รับอนุญาตไม่มีสิทธิ์ในโค้ดต้นฉบับของ API เว้นแต่ผู้รับอนุญาตจะได้รับสิทธิ์โดยชัดแจ้งโดยข้อตกลงฉบับนี้ ผู้รับอนุญาตไม่มีสิทธิ์ในการประมวลผล แยกส่วน ปรับเปลี่ยน หรือถอดการประกอบ API หรือทำให้ API อยู่ในรูปแบบที่ยอมรับโดยทั่วไปของการส่งด้วยวิธีอื่นใด ไม่ว่าทั้งหมดหรือบางส่วน ยกเว้นตามที่ได้รับอนุญาตโดยชัดแจ้งโดยข้อตกลงฉบับนี้หรือกฎหมายที่บังคับใช้
    • เพื่อการใช้งาน API ที่มีประสิทธิภาพ อาจจำเป็นต้องใช้เครื่องมือพัฒนาเพิ่มเติมอย่างเช่น คอมไพเลอร์หรือซอฟต์แวร์อื่นๆ (“ซอฟต์แวร์ภายนอก”) ผู้รับอนุญาตมีความรับผิดชอบทั้งหมดในการจัดซื้อซอฟต์แวร์ดังกล่าวและใบอนุญาตที่จำเป็น CCS Thammasat ไม่อยู่ภายใต้พันธกรณีใดๆ และไม่รับประกันเกี่ยวกับการใช้ซอฟต์แวร์ภายนอกของผู้รับอนุญาต
    • ผู้รับอนุญาตไม่มีสิทธิ์ในการให้สิทธิ์ต่อสำหรับสิทธิ์ของตนภายใต้ข้อตกลงฉบับนี้ ยกเว้นในกรณีที่ระบุไว้โดยชัดแจ้งในข้อตกลงฉบับนี้ ผู้รับอนุญาตไม่สามารถใช้เครื่องหมายการค้าหรือชื่อทางการค้าของ CCS Thammasat ได้ CCS Thammasat สงวนสิทธิ์ทั้งหมดที่ไม่มอบให้โดยชัดแจ้งในที่นี้
    • ผู้รับอนุญาตไม่สามารถขอรับสิทธิบัตรของ APIs หรือแอปพลิเคชั่นที่ถูกพัฒนาโดยใช้ API ของ CCS Thammasat หรือ API ของผู้รับอนุญาตอื่นๆ ของ CCS Thammasat เพื่อสร้าง ใช้หรือขายผลิตภัณฑ์หรือเทคโนโลยีใดๆ
  4. ทรัพย์สิน
    • CCS Thammasat จะสงวนสิทธิ์ในทรัพย์สิน รวมทั้งสิทธิ์ทั้งหมดที่มีต่อสิทธิบัตร ลิขสิทธิ์ ความลับทางการค้า เครื่องหมายการค้า และสิทธิ์ในทรัพย์สินทางปัญญาอื่นๆ ต่อ API และการเปลี่ยนแปลงใดๆ ที่มีต่อที่กล่าวมา CCS Thammasat มีสิทธิ์พิเศษแต่เพียงผู้เดียวในการจดทะเบียนสิทธิบัตรและลิขสิทธิ์ใดๆ ผู้รับอนุญาตรับทราบว่าภายใต้ข้อตกลงฉบับนี้ ผู้รับอนุญาตไม่มีสิทธิ์ในทรัพย์สินต่อ API โดยมีสิทธิ์ในการใช้งานที่จำกัดภายใต้เงื่อนไขของข้อตกลงฉบับนี้เท่านั้น
  5. การรักษาความลับ
    • API ประกอบด้วยข้อมูลที่มีค่าเกี่ยวกับเว็บไซต์และความลับทางการค้าที่เป็นของ CCS Thammasat ซึ่งยังคงเป็นทรัพย์สินของ CCS Thammasat ผู้รับอนุญาตจะต้องหลีกเลี่ยงการเปิดเผยข้อมูลดังกล่าวและหลีกเลี่ยงการใช้ API โดยไม่ได้รับอนุญาต
    • ผู้รับอนุญาตต้องไม่เปิดเผย โฆษณา หรือเผยแพร่เงื่อนไขของข้อตกลงฉบับนี้ โดยปราศจากคำยินยอมเป็นลายลักษณ์อักษรล่วงหน้าจาก CCS Thammasat การแถลงข่าวหรือการเผยแพร่ใดๆ ที่เกี่ยวข้องกับข้อตกลงฉบับนี้จะต้องได้รับการพิจารณาล่วงหน้าและการอนุมัติเป็นลายลักษณ์อักษรจาก CCS Thammasat
  6. ไม่มีการรับประกัน
    • API และเอกสารประกอบได้รับการจัดหาให้ "ตามที่เป็น" โดยไม่มีการรับประกันใดๆ ทั้งสิ้น CCS Thammasat ไม่รับประกันว่า API และเอกสารประกอบจะเหมาะสำหรับการใช้งานโดยผู้รับอนุญาต และปราศจากข้อบกพร่องหรือข้อผิดพลาด นอกจากนี้ CCS Thammasat ยังไม่รับประกันการเข้าถึง API และเอกสารประกอบได้อย่างต่อเนื่องและไม่หยุดชะงัก และไม่ทำการรับประกันหรือสำแดงใดๆ เกี่ยวกับผลของการใช้งานดังกล่าว
  7. การจำกัดความรับผิด
    • ผู้รับอนุญาตรับผิดทั้งหมดสำหรับความเสียหายใดๆ ต่อระบบคอมพิวเตอร์ของตนหรือการสูญเสียข้อมูลอันเป็นผลจากการดาวน์โหลดหรือการใช้ API CCS Thammasat จะไม่รับผิดสำหรับความเสียหายพิเศษ โดยบังเอิญ หรืออันเป็นผลตามมาใดๆ (รวมทั้งแต่ไม่จำกัดเฉพาะการสูญเสียผลกำไร การพักหรือการสิ้นสุดงาน การไม่ทำงานหรือการชำรุดของคอมพิวเตอร์ การสูญเสียข้อมูลหรือความเสียหายอื่นใด รวมทั้งความเสียหายด้านการเงิน อันเกิดจากการใช้หรือความไม่สามารถในการใช้ API หรือการให้หรือการไม่ให้บริการสนับสนุน) แม้ว่า CCS Thammasat จะได้รับแจ้งถึงความเป็นไปได้ที่จะเกิดความเสียหายดังกล่าวแล้วก็ตาม
  8. เงื่อนไขและการสิ้นสุดข้อตกลง
    • ในกรณีที่ผู้รับอนุญาตไม่ปฏิบัติตามเงื่อนไขข้อใดก็ตามของข้อตกลงฉบับนี้ ผู้รับอนุญาตจะต้องรับผิดต่อ CCS Thammasat สำหรับความสูญเสียหรือความเสียหายใดๆ อันเกิดจากการละเมิดเงื่อนไข
    • เมื่อสิ้นสุดข้อตกลงฉบับนี้ ผู้รับอนุญาตจะต้องยุติการใช้ API ลบสำเนาทั้งหมดของ API และเอกสารประกอบ หรือส่งคืนให้แก่ CCS Thammasat โดยทันที
    • CCS Thammasat จะต้องมีสิทธิ์ตรวจสอบการปฏิบัติตามเงื่อนไขของข้อตกลงของผู้รับอนุญาต ผู้รับอนุญาตต้องอำนวยความสะดวกให้ดำเนินการดังกล่าวตามการร้องขอจาก CCS Thammasat
  9. เบ็ดเตล็ด
    • การฝ่าฝืนหรือการไม่ปฏิบัติตามเงื่อนไขของข้อตกลงฉบับนี้จะส่งผลให้สิ้นสุดข้อตกลง
    • CCS Thammasat จะต้องมีสิทธิ์ระงับ (หรือสิ้นสุด) สิทธิ์เข้าถึงบริการของผู้รับอนุญาต ภายใต้ดุลยพินิจของตนเองและไม่มีการบอกแจ้ง ในกรณีที่มีการฝ่าฝืนเงื่อนไขของข้อตกลง
การติดตั้ง การคัดลอกหรือการใช้ API ของ CCS Thammasat ในทางอื่นใด จะถือว่าผู้รับอนุญาตรับทราบว่าตนได้อ่าน เข้าใจ และตกลงตามเงื่อนไขที่ระบุข้างต้นแล้ว

ข้าพเจ้ายอมรับเงื่อนไขและข้อตกลงในการใช้บริการข้อมูลผ่าน API ของเว็บไซต์ CCS Thammasat
